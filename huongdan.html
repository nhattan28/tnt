Định Dạng File Excel

Để trang website có thể đọc dữ liệu chính xác, file Excel của bạn phải có tối thiểu ba cột với các tiêu đề sau:

Mã Lớp Số ĐVHT Điểm gốc
CS 201 HF1 2 8.5
EVR 205 BJ 2 8.9
CHE 101 D1 3 8.0

Lưu ý: Bạn có thể đặt các cột/hàng này ở bất kỳ vị trí nào trong file Excel, nhưng tiêu đề phải chính xác để trang website nhận diện được.
